> Royal Task Ticker — enter tasks in `HH:MM Description` format, get timed reminders, earn coins for completion, compete on the leaderboard. 1-hour focus mode blocks distractions. Android WebView bridge included.

---

## ✨ Features

- Timed and general tasks (`9:00 Morning run`)
- Automatic reminders at scheduled times
- 1-hour Working Mode (focus/distraction blocker)
- Coin animations and sound effects
- Leaderboard gamification
- Daily task reset
- Android WebView bridge
- PWA manifest

---

## 🚀 Quick Start

```bash
git clone https://github.com/maximosovsky/MaxTaskX.git
cd MaxTaskX
npm install
node server.js
```
